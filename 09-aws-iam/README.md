# Module 09: AWS IAM Deep Dive

## Overview
Master AWS Identity and Access Management: policies, roles, federation with Auth0, cross-account access, IAM Identity Center, and security best practices.

## Topics
1. IAM Users, Groups, Roles, Policies
2. IAM Policy Language (JSON) deep dive
3. Least Privilege principle in practice
4. Roles for EC2, Lambda, ECS, SageMaker
5. Cross-account access with AssumeRole
6. SAML Federation with Auth0
7. OIDC Federation (GitHub Actions)
8. AWS IAM Identity Center (SSO)
9. Service Control Policies (SCPs)
10. IAM Access Analyzer
11. Permission Boundaries
12. AWS STS
